# TalentOps - Desarrollo Full Stack

Este repositorio contiene los distintos ejercicios y proyectos desarrollados como parte del programa **TalentOps**, con el objetivo de fortalecer mis conocimientos en **Desarrollo de Software**, especialmente en **JavaScript**.

## Objetivo del aprendizaje

- Reforzar los fundamentos del lenguaje **JavaScript**.  
- Aplicar buenas prácticas de programación.  
- Comprender el flujo de trabajo con **Git** y **GitHub**.  
- Desarrollar una base sólida para avanzar en tecnologías de desarrollo web y backend.
